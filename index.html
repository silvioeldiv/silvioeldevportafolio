Sistema de Gestión de Tareas

Un sistema de gestión de tareas que permite a los usuarios crear, actualizar y eliminar tareas. Los usuarios pueden categorizar sus tareas, establecer fechas de vencimiento y marcar tareas como completadas. La aplicación ofrece una interfaz de usuario intuitiva y una API REST para interactuar con el sistema desde aplicaciones externas.

Plataforma de Reservas de Eventos

Una plataforma que permite a los usuarios buscar y reservar eventos locales, como conciertos, conferencias y talleres. Los organizadores pueden listar sus eventos y gestionar las reservas. La aplicación proporciona una API REST para la integración con aplicaciones móviles y una interfaz web para la administración.
